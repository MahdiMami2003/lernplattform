Benutzeranleitung für den Endkunden:

Allgemeines:

Zugangsdaten Erhalten:
Um Zugangsdaten zu erhalten einmal auf der Hauptseite auf "Noch keine Nutzerdaten? Hier gehts zur Registrierung" klicken.  Dort die Felder ausfüllen, Rolle auswählen und die AGB akzeptieren. Dann auf registrieren klicken. Hinweis: Daten aufschreiben, da die Daten nicht zurückgesetzt werden können.

Anmelden: 
Mit den Anmeldedaten von der Registrierung die Felder auf der Anmeldeseite ausfüllen und auf "Einloggen" klicken.

Abmelden: 
Wenn angemeldet, oben rechts auf das Logout-Symbol drücken.

Layout bedienen:
Links kann man indem man auf das gewünschte Symbol klickt direkt zu dem gewünschten bereich kommen.  Oben rechts findet man Logout, Landing-Page (Haupt Seite für die Person, sieht je nach Rolle unterschiedlich aus) und die Suchfunktion. Bei Klick auf das Symbol oder den Namen der Schule oben rechts kommt man wieder auf die Landing-Page. Ganz unten findet man Links zu dem Impressum, dem Datenschutz und das Formelle zur Barrierefreiheit. Barrierefreiheit findet man auf der Linken Seite.

Schüler-innen - Bereich:

Lernunterlagen ansehen und Herunterladen:
Auf der Schüler-Seite auf "Lernunterlagen" klicken dort ein Thema auswählen und anklicken. Nun findet man eine Seite wo oben der Lehrer einen Text bereitgestellt hat, und darunter ein Feld mit einer PDF die der Lehrer hochgeladen hat. Darunter kann man mit den Knopf "PDF herunterladen" die Datei downloaden.

Eigenen Fortschritt für den Spielebereich sehen:
Einfach auf der Schüler-Seite auf "Fortschritt" klicken. Dort findet man alle Fortschritte.

Wöchentliche Tests ansehen und Herunterladen:
Auf der Schüler-Seite auf "Wöchentliche Tests" klicken dort eine Überschrift auswählen und anklicken. Dort wird nun eine PDF angezeigt welche der Schüler eigenständig bearbeiten kann. Darunter ist ein Knopf "Lösung Anzeigen" wo der Schüler sich die Lösung für Eigenkorrektur ansehen kann. Beide PDFs können per Klick auf den grünen Knopf gedownloadet werden

Spiele Bereich:
Wenn man auf der Schüler-Seite auf "Game Page" klickt, werden einem die Fächer angezeigt für die man Spiele spielen kann. Darüber ist ein Knopf wo man den Lückentext modus ein und ausschalten kann. Wenn man auf ein Fach gedrückt hat, kann man das Thema auswählen. Durch richtiges Klicken auf die richtige Antwort oder durch Eingabe der richtigen Lösung kann man die Spiele durch spielen.

KI-Tool:
Nach einem Klick auf "Lernabenteuer mit KI-Robo" kann man Fach, Thema, Klasse und Schwierigkeitsstufe einstellen. Wenn man auf generieren Klickt, dauert es einige Sekunden bis die Aufgaben generiert und geladen werden. Nun werden einem die generierten Fragen angezeigt. Als Schüler kann man sich diese dann ansehen und eine Lösung überlegen. Um die korrekte Antwort anzuzeigen, drückt man auf Lösung Anzeigen. Zusatzfeature: wenn man die Fragen erstellt während man im englischen Modus ist, werden die Fragen auf englisch generiert.



Eltern:

Lernunterlagen und Fortschritt des Kindes:
Selbe Struktur wie bei den Schülern selbst

Pädagogische Tipps:
Wenn man auf der Eltern-Seite auf "Pedagogische Tipps" klickt, werden einem diese angezeigt. Man kann dann auf das gewünscht Theme klicken und die von den Lehrern bereitgestellten Inhalte durchlesen.

Termine und Aktuelles: 
Einfach auf den Link "Termine und Aktuelles" klicken. Dort werden einem die Termine angezeigt.


Lehrer:

Hinzufügen von Inhalten:
Es gibt 3 Möglichkeiten um es so angenehm wie möglich zu gestalten:
1.  Auf der Lehrer Landing Page unter "Neue Inhalte erstellen" das gewünschte Formular auswählen und auf erstellen klicken. Dort das Formular ausfüllen
2.  Auf der Lehrer Landing Page unten bei den zugeteilten Klassen das gewünschte Formular anklicken  und dort das Formular ausfüllen.
3. Auf der Übersichtsseite des Bereichs (bspw Lerninhalte) auf den dafür vorgesehenen Knopf klicken und dort das Formular ausfüllen.

Löschen von Inhalten:
Auf der Übersichtsseite des Bereichs (bspw Lerninhalte) auf den Knopf "Löschen" klicken und bestätigen.

Verwaltung/Administrator:

Auf dem Admin Dashboard können neue Klassen hinzugefügt werden, oder gelöscht werden.
Dort können die Rollen aller Personen bearbeitet werden, sowie rechte für Bearbeitung vergeben werden.
